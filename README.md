# LOL_MOTD

Get cool motd with mastery score on your favorit champion :o \
It write the motd to `/etc/motd.d/10-header`

## requirements

You need figlet


## Environment variables

- `$LOL_MOTD_REGION`
  - default: euw1
- `$LOL_MOTD_SUMMONER`
  - Summoner puuid
- `$RIOT_KEY`
  - Riot api key
- `$LOL_MOTD_CHAMP`
  - Champion name
  - Case sensitive
