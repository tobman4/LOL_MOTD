# LOL_MOTD                             
League of Legends motd                 
                                       
## Environment variables

- `$LOL_MOTD_REGION`
  - default: euw1
- `$LOL_MOTD_SUMMONER`
  - Summoner id
- `$RIOT_KEY`
  - Riot api key
- `$LOL_MOTD_CHAMP`
  - Champion name
  - Case sensitive
